# mark-vision

CodeMirror6 plugin that supports hybrid markdown mode like HyperMD

## Features

- [x] Emphasis
  1.  Bold: `**`, `__`
  2.  Italic: `*`, `_`
  3.  Strikethrough: `~~`
  4.  Underline: `--`
  5.  Highlight: `==`
- [x] Inline Code
- [x] Escape
- [x] Heading
  1.  Normal Heading: `#`
  2.  SetextHeading: `===`, `---`
- [x] HashTag
- [x] Mention
- [x] Bullet List
- [x] Task List
- [x] Ordered List
- [x] Image
- [x] Link
- [x] HorizontalRule
- [ ] Footnote
- [ ] CodeBlock
- [ ] Quote
- [ ] GFM Alert
- [ ] Front matter
- [ ] LaTeX
- [ ] TikZjax
- [ ] Table
